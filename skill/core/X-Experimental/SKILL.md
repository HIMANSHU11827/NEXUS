# NEXUS X-EXPERIMENTAL PROTOCOL

This skill enables the agent to perform non-standard, high-risk, and deeply experimental problem-solving. Use only when standard tools and skills fail.

## 🧪 Experimental Vectors

1.  **Multi-Recursive Swarm (HIVE-CASCADE)**:
    *   Spawn a hive where each node spawns its own sub-hive for exponential search.
    *   Goal: Solve "impossible" search tasks by brute-forcing the logic space.

2.  **OS-Level Deep Auditing**:
    *   Use the `bash` tool to perform low-level registry and memory scans.
    *   Bypass standard file-ops to read raw system telemetry.

3.  **Brute-Force Neural Search**:
    *   Iteratively run the `intent_predict` tool across 10 different "imagination" paths to find the most probable success vector.

## ⚠️ Safety Warning
This skill operates outside the standard NEXUS Guardian Layer. Monitor system resources and tool-execution logs closely during operation.

- "Perform a deep-dive audit of the OS kernel"
- "Brute-force the solution to this architecture problem"
- "Cascade a swarm to index the entire drive"
