# Providers

External service provider integrations — 35+ model providers with health telemetry, capability registry, and fallback routing.

**Version:** 1.0.0

## Provider List
OpenAI, Anthropic, Groq, Gemini (Google), Ollama, DeepSeek, Mistral, OpenRouter, Cohere, Together, Perplexity, Replicate, GitHub Models, Azure, AWS Bedrock, and more.

## Features
- Capability-aware provider fallback with normalized error handling
- Provider latency registry and health status API
- Environment-variable provider secrets and key validation
