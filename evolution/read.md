# Evolution

NEXUS AI self-evolution system — forges, learning, and improvement.

## Modules
- `forge/` — ToolForge: auto-create tools
- `skill_forge/` — SkillForge: auto-create skills
- `plugin_forge/` — PluginForge: auto-create plugins
- `memory_forge/` — MemoryForge: create memory entries
- `knowledge_forge/` — KnowledgeForge: create knowledge entries
- `log.py` — EvolutionLog: session logging
- `log_analyzer.py` — LogAnalyzer: pattern analysis
- `ledger/` — EvolutionLedger: immutable record
- `status.py` — EvolutionStatus: system health
- `nudge/` — NudgeEngine: behavioral nudges
- `intent/` — NexusIntentEngine: intent evolution
- `self_improvement/` — SelfImprovementEngine: session analysis
- `sop/` — Standard Operating Procedures
- `ensemble/` — EnsembleManager: multi-strategy reasoning
- `skill_synthesizer/` — SkillSynthesizer: generate skills from examples
